GTT PIRATE API

Anche i nerd prendono il bus...

 

Questa è la GTT Pirate API, una API non ufficiale (e probabilmente anche illegale) per accedere alle informazioni sui passaggi in tempo reale dei mezzi di trasporto pubblici attivi a Torino.

Per interrogare l'API, invoca l'endpoint 

        http://gpa.madbob.org/query.php?stop=XXX

dove XXX è il numero identificativo di una fermata.

Riceverai un file JSON recante alcune informazioni sulle linea in prossimo transito presso la fermata stessa, così strutturato: 

        [
          {
                  "line": "13",
                  "hour": "00:40",
                  "realtime": "true",
          },
          {
                  "line": "13",
                  "hour": "01:05",
                  "realtime": "true",
          }
        ]

Niente di più, niente di meno. Rapido ed indolore. Vien da chiedersi perché 5T non abbia già provveduto ad erogare tale banale servizio, ed un povero nerd si debba mettere a smanettare un'ora per cavare qualcosa...
